核心特色

01 · GENERATION

程序生成地形

從 Square grid 出發，配上圓形 mask、雙層 sine 波 curve distortion、shape-aware smoothing，產出邊長近等距、外輪廓圓潤的 100% quad 格網。

技術堆疊

THREE.JS 0.160

3D 渲染核心 — WebGL、PCF 軟陰影、ACES Filmic Tonemapping、OrbitControls、PerspectiveCamera。

DELAUNATOR 5

不規則格網生成輔助：Delaunay 三角剖分演算法，本作改用 square grid + jitter，但仍保留 utility 給後續實驗。

THREE-SUBDIVIDE

Loop subdivision 實作：每個三角形分成 4 個並做 smoothing，把硬邊的 block geometry 圓潤成 Townscaper 風的 puffy 形狀。

GLSL SHADER

四套自訂 shader — 天空漸層球 + FBM 雲、徑向漸層水面、屋頂瓦片圓點 dot、牆面水平條紋。

POST-PROCESSING

EffectComposer + UnrealBloomPass：strength 0.35 / threshold 0.85 的微暈光，增加溫暖夢幻感。

ES MODULE + IMPORT MAP

零 build：純原生 ES Module 從 CDN 載入；只要靜態 server 就能跑，無 webpack / vite。

功能亮點

  • 兩階段建造：第一次點空地 = 矮地基（從水下到水面 +0.3）；之後 = 一格高樓層 + 斜屋頂。
  • 屋頂自動形變：依「region 內部度」決定該頂點抬升程度，孤立格成金字塔、雙格成 gable、大叢集成平頂。
  • InstancedMesh 門窗：每棟自動選最長外牆放拱形門，其他外牆依長度放 1–2 扇拱窗。
  • 屋簷 overhang：頂層屋簷沿未被鄰格阻擋的邊外推，避免階梯建築穿模。
  • 無限 Undo：history snapshot 堆疊 100 步，⌘Z / Ctrl-Z 可隨時退回。
  • Hover Ghost：3D 半透明預覽方塊含呼吸動畫與 footprint 提示，地基模式與樓層模式視覺不同。
  • 平滑追焦：camera 與 OrbitControls target 隨建築重心緩慢 lerp 移動，保留使用者視角。
  • 後處理 Bloom：UnrealBloom threshold=0.85 只挑亮部，營造微暖暈光。
  • 自動繞鏡頭：開始選單下 camera 繞地圖中心緩慢 orbit，預覽用示範小鎮。
  • cinematic 選單：左側漸層黑 + monospace 選項，支援 ↑/↓/Enter 鍵盤導航。

開發歷程

STAGE 01

地形系統

從 hex 格網轉到 square + jitter；加入 curve distortion；用 shape-aware smoothing 收斂邊長。產出 441 個 quad cells。

STAGE 02

渲染管線

每棟建築合併 geometry → Loop subdivision → vertex colors 烤入；onBeforeCompile 注入瓦片與木紋 shader。

STAGE 03

裝飾系統

屋頂依 region 內部度自動形變、屋簷 overhang 避開鄰格穿模、InstancedMesh 門窗自動安置。

STAGE 04

UI / 後處理

UnrealBloom 微暈光、自訂天空 shader、cinematic 開始選單、鍵盤導航、PROJECT INFO 技術導覽頁。
